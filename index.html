Why is a Transfer Code Required?

To protect investors' funds from unlawful withdrawals, a transfer code is required. This is in accordance with financial regulations to ensure the security and legitimacy of each transaction. The transfer code acts as an added layer of verification to safeguard both the user and the platform from unauthorized transactions. It is a precautionary measure to protect all parties involved in the transaction, in compliance with anti-money laundering (AML) and combating the financing of terrorism (CFT) laws.

We appreciate your understanding and cooperation in following these security protocols.

Reference: Financial Conduct Authority (FCA) - Withdrawal Policies & Security Measures.
